<h1 id="generate-csr-certificate" data-typography="Generate CSR certificate">Generate CSR certificate</h1>

Follow these steps to create and install a test certificate. Keep your private key secure, only the CSR and certificate are shared with Staq.

<ButtonNavigation to="/dashboard/certificates" label="Generate certificate" />
<div className="spacer"></div>
<h2 id="generate-a-private-key" data-typography="Generate a private key">Generate a private key</h2>

Open your terminal and create a new RSA private key. This file stays on your system keep it safe and never share it with anyone.

<InlineCodeBlock code="$ openssl genrsa -out server.key 2048 " />
<div className="spacer"></div>
<h2 id="generate-a-certificate-signing-request" data-typography="Generate a Certificate Signing Request (CSR)">Generate a Certificate Signing Request (CSR)</h2>

Use your private key to generate a CSR. This file contains your public key and organisation details, which we’ll use to issue your Sandbox certificate.

<InlineCodeBlock code="$ openssl genrsa -out server.key 2048 " />
